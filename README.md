# json-edit-vue3


## DEMO

```sh
# install
npm run pre

# for vue 3.x app
npm run dev

```


## How to use

```sh
# install
npm install json-edit-vue3 --save

# require
import jsonEdit from "json-edit-vue3";
import "json-edit-vue3/dist/style.css";
```


### Type-Check, Compile and Minify for Production

```sh
npm run build
```
